# ProjetoIntegradoLPR_SE_1TRI
Projeto desenvolvido com STM32 e display ST7735 para controle de sala de aula, permitindo registrar presença, controlar saída de alunos para o banheiro, limitar quantidade de alunos fora da sala, utilizar senha de acesso, LEDs de sinalização e gerar um relatório final da aula.
