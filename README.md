# ProjetoIntegradoLPR_SE_1TRI
Projeto desenvolvido com STM32 e display ST7735 para controle de sala de aula, permitindo registrar presença, controlar saída de alunos para o banheiro, limitar quantidade de alunos fora da sala, utilizar senha de acesso, LEDs de sinalização e gerar um relatório final da aula.

# Controle Inteligente de Sala de Aula - STM32

Projeto desenvolvido utilizando STM32, linguagem C e display TFT ST7735 para simular um sistema inteligente de controle de sala de aula.

O sistema permite:

* Registrar presença de alunos
* Controlar saída para banheiro
* Limitar quantidade de alunos fora da sala
* Utilizar senha de acesso
* Exibir informações no display TFT
* Utilizar LEDs como sinalização
* Gerar relatório final da aula

## Componentes Utilizados

* STM32
* Display ST7735
* LEDs
* Botões
* Comunicação SPI

## Tecnologias

* Linguagem C
* STM32CubeIDE
* HAL Library
